# 일관성 있는 캐릭터 영상 만들기

## 관련 자료

- 나노바나나로 애니메이션 제작: https://www.youtube.com/watch?v=AekbV54aI_g

## Google Labs

- 이미지 생성: Whisk
- 동영상 생성: Flow

### 생성 흐름

- `1. Whisk로 캐릭터 이미지 생성`

```prompt
# prompt 1: 캐릭터 1 생성
두발로 서 있는 아기 북극곰 캐릭터, 흰 배경, 3d 픽사 애니메이션 스타일, 와이드샷

# prompt 2: 캐릭터 2 생성
두발로 서 있는 아기 펭귄 캐릭터, 흰 배경, 3d 픽사 애니메이션 스타일, 와이드샷

# pormpt 3: 캐릭터에 옷 입히기
 - 이미지 클릭 후 프롬프트 입력
청색 멜빵 바지를 입고 두발로 서 있는 아기 북극곰 캐릭터, 흰 배경, 3d 픽사 애니메이션 스타일, 와이드샷

# prompt 4: 두 캐릭터에 키를 맞추기 위해 한장으로 묶음
 - 두 캐릭터를 피사체에 등록
함께 이야기를 나누는 북금곰과 펭귄 캐릭터
```

- `2. Gemini로 장면별 프롬프트 생성`

```prompt
1. 짐을 싸고
2. 공항에 가고
3. 비행기를 타고
4. 휴양지에서 노는 내용

각각 나노바나나로 이미지를 생성할거야. 프롬프트를 작성해줘 (이미지는 생성하지 말아줘)
3d 애니메이션 스타일로 만들거고, 북극곰과 펭귄 캐릭터의 생김새를 묘사하지 말아줘.
```

- `3. Whisk로 각 장면별 이미지 만들기`

```prompt
# prompt
 - 두 캐릭터가 함께 있는 이미지를 피사체로 등록
Gemini의 하나의 장면 프롬프트 입력
```

- `4. Flow로 각 장면별로 영상 만들기`

```prompt
# prompt
 - 에셋으로 동영상 만들기 -> 이미지 업로드
Gemini의 하나의 장면 프롬프트 입력 (이미지 제작 프롬프트 그대로 입력)
```

- `5. 캡컷으로 동영상 합치기`

```
1. 4개의 동영상 업로드
2. 자르고 싶은 부분 자르기 (Ctrl + B 이후 Delete)
3. 기본 사운드 없애고, 원하는 브금 넣기
```
