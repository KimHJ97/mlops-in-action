# 동물 영상 만들기

## 1. 워크 플로우

- Gemini로 이미지 생성
- Flow로 동영상 생성

## 2. 예시

- `Gemini로 이미지 생성`

```prompt
책상에 앉아 녹음을 하는 중인 아기 호랑이의 모습을 만들어줘.
```

- `Flow로 동영상 생성`
  - 무슨 캐릭터
  - 어떤 목소리

```
- 이미지 업로드 및 프롬프트 입력
호랑이 캐릭터가 3살 한국 남자아이가 목소리로 말하는 영상을 만들어줘. "여러분, 새해복 많이 받으세요!" 말이 끝나고 귀엽게 웃어줘.
```

- `Flow로 동영상 더 길게 만들기`

```
호랑이 캐릭터가 3살 한국 남자아이가 목소리로 "영상 시청해주셔서 감사합니다!"라고 말하는 영상을 추가해줘.
```

- `대화 주고 받게 만들기`

```
흰색 강아지와 인터뷰어가 대화하는 영상 만들어줘. 흰색 강아지가 3살 남자아이 목소리로 "날씨가 추워서 산책 나가는 걸 잠깐 고민했어요"라고 말하고, 인터뷰어가 "그래도 산책 나오니까 좋죠?"라고 말하면 좋겠어. 인터뷰어의 목소리는 30대 남성 목소리로 해줘. 흰색 강아지가 "네!" 라고 3살 남자아이 목소리로 대답하는 걸로 영상을 마무리해줘.
```
