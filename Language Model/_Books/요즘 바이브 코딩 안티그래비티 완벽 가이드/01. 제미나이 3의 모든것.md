# 제미나이 3의 모든것

## 1. 제미나이 3 이해하기

제미나이 3는 MoE 기반의 트랜스포머 아키텍처를 사용한다. 멀티모달 AI 시스템이며, 업계 최대 규모의 콘텍스트 윈도우를 가진다. 강력한 추론 능력을 위해 CoT 기법을 사용한다.

 - __CoT__
    - CoT 기법은 복잡한 문제를 단계별로 분해하여 해결하는 방식을 말한다. 수학 문제나 알고리즘 설계와 같이 논리적 사고가 필요한 작업에서 이 방식이 활용된다. 모델은 중간 추론 단계를 명시적으로 생성함으로써 최종 답안의 정확도를 높인다.
 - __멀티모달__
    - 멀티모달은 멀티-모달리티의 합성어로 다양한 형태의 정보를 뜻한다.
    - 제미나이 3는 텍스트, 이미지, 비디오, 오디오 등 다양한 형태의 데이터를 동시에 처리할 수 있다.
    - 텍스트, 이미지, 오디오, 비디오 등 서로 다른 종류의 데이터를 각각 전용 인코더로 처리한다. 인코더는 원본 데이터를 AI가 이해할 수 있는 숫자 형태로 바꾸는 장치이다.
 - __MoE__
    - MoE는 원래 여러 전문가를 두고 조합해서 쓰는 방식을 말하며, MoE 방식 중 상황에 적절한 전문가만 선택하는 방식을 Sparse MoE라고 부른다.
    - MoE로 일하는 방식을 설명하자면, 모든 것을 다 아는 하나의 천재에게 일을 맡기는 대신, 각 분야의 전문가로 구성된 팀을 만들고 상황에 따라 적합한 전문가 몇 명에게만 일을 시켜 성능을 올린다.
 - __트랜스포머__
    - 트랜스포머는 현대 LLM의 기반이 되는 핵심 아키텍처로 기존 방식과 달리 입력 문장 전체를 동시에 고려해 처리한다.
    - 기존에 '나는 사과를 좋아한다'라는 문장이 있을 때 단어를 순차적으로 처리했다면, 트랜스포머는 문장 전체 구조 속에서 '사과'와 '좋아한다'의 관계를 병렬로 계산한다.
    - 이런 문맥 정보를 바탕으로 다음에 올 가장 적절한 단어를 확률적으로 예측하며 답변을 생성하는 것이 트랜스포머 기반 모델의 특징이다.
 - __콘텍스트 윈도우__
    - 제미나이 3의 콘텍스트 윈도우는 2025년 11월 기준 1M, 최대 출력은 64K 토큰으로 업계 최대 규모의 콘텍스트 윈도우를 제공한다.
    - __콘텍스트 윈도우는 모델이 한 번에 처리할 수 있는 입력 데이터의 양을 의미한다.__
        - `장기 대화 유지`: 이전 대화 내용을 모두 기억하면서 일관성 있는 응답을 생성
        - `대규모 코드베이스 분석`: 여러 파일에 걸친 코드 구조를 한 번에 이해
        - `문서 전체 이해`: 긴 기술 문서나 사양서를 전체적으로 파악하여 질문에

## 2. 제미나이를 사용하는 다양한 방법

제미나이를 사용하는 대표적인 방법으로는 제미나이 웹, 구글 AI 스튜디오, 구글 클라우드 버텍스 AI, 안티그래비티 등이 있다.

 - __제미나이 웹__
    - URL: https://gemini.google.com
    - 제미나이 최신 모델을 사용해서 텍스트, 이미지, 동영상 등을 생성할 수 있다.
    - 구글의 최신 비디오 생성 엔진인 Veo 3도 간단히 사용할 수 있어 가벼운 작업에 유리하다.
 - __구글 AI 스튜디오__
    - URL: https://aistudio.google.com
    - 제미나이 웹 인터페이스를 조금 더 전문적인 작업에 적합하게 변형한 형태
    - Temperature, Media Resolution, Thinking Level 조정, Code Execution을 통한 복잡한 작업 자동화 등 세세한 컨트롤 가능
 - __구글 클라우드 버텍스 AI__
    - URL: https://console.cloud.google.com/vertex-ai/dashboard
    - 구글 클라우드 버텍스 AI는 독립적인 사이트로 운영하지 않고 구글 서비스 통합 플랫폼인 구글 클라우드 플랫폼에서 제공한다.
    - 머신러닝 모델을 커스터마이즈하고 생성형 AI를 파인튜닝하여 배포할 수 있다.
