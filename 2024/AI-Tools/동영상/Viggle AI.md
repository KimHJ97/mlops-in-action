# Viggle AI

 - 공식 홈페이지: https://viggle.ai/
 - 유튜브 영상: https://www.youtube.com/watch?v=lzbVtZwkU3w

<br/>

## 사용 커맨드

 - /animate
    - 레퍼런스 사진과 비글 AI에서 제공해주는 모션을 부여
```
1. 이미지 업로드
2. 원하는 모션 프롬프트를 찾아서 입력
3. 크로마키 합성을 원하는 경우 > 백그라운드 색상 선택
```
<br/>

 - /mix
    - 레퍼런스 사진에서 레퍼런스 영상의 모션을 기반으로 영상 생성
```
1. 이미지 업로드
2. 모션 동영상 업로드
3. 크로마키 합성을 원하는 경우 > 백그라운드 색상 선택
```
